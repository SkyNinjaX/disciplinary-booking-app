# 📋 Disciplinary Booking System

A role-based web application for tracking student disciplinary records, built with Flask and SQLite.

## ✨ Features

- 🔐 **Role-Based Access Control**: Admin and Teacher roles with different permissions
- 📝 **CRUD Operations**: Add, view, and delete disciplinary records
- 🚨 **Auto-Detention Flag**: Automatically flags students with repeated offences
- 🔍 **Filtering**: Search by student name, class, or date
- 🔄 **Live Refresh**: Auto-refreshes every 30 seconds + manual refresh button
- 📱 **Responsive Design**: Built with Tailwind CSS

## 🛠 Tech Stack

- **Backend**: Python (Flask)
- **Database**: SQLite
- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Auth**: Predefined credentials (Demo)
